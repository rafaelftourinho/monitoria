# Instalação e uso

```bash
# Entre no site do Vite e crie o projeto
npm create vite@latest

# Entre na pasta web com criada com o nome escolhido
cd hooks

# Instale as dependências
yarn install
ou 
npm install

# Rode o aplicação
npm run dev 

# Acesse http://localhost:5173/ no seu navagador.
```
